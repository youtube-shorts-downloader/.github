# .github
Uniloader — это Telegram-бот и надёжный Youtube shorts downloader, который помогает быстро получить короткие ролики прямо в чате. Просто отправьте ссылку, бот выполнит обработку и вернёт результат. Всё происходит мгновенно, без регистрации и переходов на сайты. Работает на всех устройствах.
